# Rick And Morty 

### Instructions:
* Run "composer install" to get the required packages.

* Start a local PHP server. (Run: "php -S localhost:8000" in RickAndMorty/public/ directory.)

* Go to http://localhost:8000/, enjoy.

### Technologies:
* PHP 7.4
* Twig
